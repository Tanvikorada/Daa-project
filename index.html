How Backtracking Works

What is Backtracking?

Backtracking is a recursive algorithm that systematically tries every possible path. Like navigating a maze — when you hit a dead end, retrace your steps and try a new direction.

This algorithm tries every route from S(0,0) to E(N-1,N-1), skipping mines, and records the shortest safe path found.

Pruning — The Power Move

If the current path is already longer than the best path found so far, the algorithm immediately abandons that branch. This "pruning" avoids millions of pointless routes.

Key Properties

  • Moves in 4 directions: up, down, left, right
  • Marks cells visited to avoid cycles
  • After exploring a branch, unmarks cells so other paths can use them
  • Guarantees the shortest path by exhaustive search

Algorithm Step-by-Step

1
Start at (0,0). Mark visited. Add to current path.
2
Check 4 neighbors. In bounds? Not a mine? Not already visited in THIS branch?
3
Prune. If path length ≥ best found, stop exploring this branch.
4
Recurse. Move to valid neighbor and repeat from step 1.
5
Goal reached! Update best path if this one is shorter.
6
Backtrack. Remove current cell from path. Try next direction from parent.
7
Done when all branches exhausted. Best path is the answer.

Pseudocode

SOLVE(grid, N): bestPath ← [] function BACKTRACK(row, col, path, visited): ▸ Prune invalid / already-visited if out-of-bounds OR mine OR visited → return if len(path)+1 ≥ len(bestPath) > 0 → return ▸ Choose this cell visited.add(row, col) path.append((row, col)) if (row,col) == (N-1,N-1): bestPath ← copy(path) ▸ New best! else: for (dr,dc) in [UP,DOWN,LEFT,RIGHT]: BACKTRACK(row+dr, col+dc, path, copy(visited)) ▸ Unchoose (backtrack!) path.pop() BACKTRACK(0, 0, [], {}) return bestPath

Time & Space Complexity

TIME WORST CASE
O(4^N²)
4 branches per cell
SPACE STACK
O(N²)
Max recursion depth

With Pruning (Real World)

  • Mines eliminate whole subtrees instantly
  • Length bound cuts suboptimal branches
  • Actual perf far exceeds worst-case theory
  • On 8×8 with 20% mines: typically <500 steps!
